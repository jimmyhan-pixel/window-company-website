## 🚀 如何使用

### **Step 1: 确保密码已设置**

打开 `.env.local`，确认已设置：
```bash
DASHBOARD_USERNAME=admin
DASHBOARD_PASSWORD=[你的密码]
```

### **Step 2: 访问 Dashboard**

1. 开发服务器已在运行: http://localhost:3000
2. 访问登录页面: http://localhost:3000/dashboard/login
3. 输入用户名和密码
4. 登录成功后进入 Dashboard

### **Step 3: 测试完整流程**

1. **提交测试询价**
   - 访问: http://localhost:3000/quotation
   - 填写并提交表单

2. **在 Dashboard 查看**
   - 登录 Dashboard
   - 应该能看到新提交的询价
   - 状态为 "🟡 新询价"

3. **更新状态**
   - 点击询价记录的"更新"按钮
   - 选择新状态（已报价/已关闭）
   - 添加备注（可选）
   - 点击"保存"

4. **筛选查看**
   - 点击顶部统计卡片或筛选按钮
   - 查看不同状态的询价

## 📊 Dashboard 功能详解

### **统计卡片**
```
📩 新询价    - 显示 status='new' 的数量，点击筛选新询价
📧 已报价    - 显示 status='quoted' 的数量，点击筛选已报价
✅ 已关闭    - 显示 status='closed' 的数量，点击筛选已关闭
📈 转化率    - 已关闭 / 总询价 × 100%
```

### **询价列表**
显示字段：
- **ID**: 询价唯一标识（前8位）
- **客户信息**: 邮箱 + 电话
- **窗户配置**: 类型、材料、尺寸、数量、颜色
- **状态**: 带颜色的状态标签
- **时间**: 相对时间（"2小时前"）
- **操作**: 更新按钮

### **状态管理**
- **new (新询价)**: 🟡 黄色 - 刚提交，待处理
- **quoted (已报价)**: 🔵 蓝色 - 已通过邮件发送报价
- **closed (已关闭)**: 🟢 绿色 - 已成交或放弃

## 🎯 工作流程示例

### **典型的一天**

```
早上 9:00
├─ 访问 http://localhost:3000/dashboard/login
├─ 输入用户名和密码登录
├─ 看到 "新询价: 3" 
├─ 点击 "新询价" 卡片
└─ 查看 3 条新询价记录

处理询价
├─ 点击第一条记录的"更新"
├─ 查看客户信息和窗户配置
├─ 复制客户邮箱: customer@email.com
├─ 在邮箱客户端回复报价
├─ 回到 Dashboard
├─ 点击"更新" → 选择"已报价"
├─ 添加备注: "报价 $1500，等待回复"
└─ 点击"保存"

下午 3:00 - 客户确认
├─ 客户电话确认接受报价
├─ 在 Dashboard 找到该询价
├─ 点击"更新" → 选择"已关闭"
├─ 添加备注: "客户接受，预计下周安装"
└─ 点击"保存"

周五下午 - 查看统计
├─ 查看统计卡片
├─ 本周: 15 个询价，3 个成交
└─ 转化率: 20%
```

## 🔄 数据流程

```
客户提交表单 (quotation page)
    ↓
保存到 Supabase (quotes 表)
    ↓
发送邮件 (Resend)
    ↓
Dashboard 显示新询价 (status='new')
    ↓
管理员查看并回复邮件
    ↓
在 Dashboard 更新状态为 'quoted'
    ↓
客户确认
    ↓
更新状态为 'closed'
```
