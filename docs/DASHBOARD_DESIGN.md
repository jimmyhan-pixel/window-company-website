# 📊 Dashboard 功能设计方案

## 🎯 核心目标

为窗户公司创建一个**简单、实用、美观**的询价管理 Dashboard，帮助你：
1. 快速查看和管理所有询价
2. 了解业务数据和趋势
3. 提高工作效率

---

## 🏗️ 页面结构

```
/dashboard
├── 顶部统计卡片（Overview）
├── 筛选和搜索栏
├── 询价列表（主要内容）
└── 详情侧边栏（可选）
```

---

## 📋 功能模块详解

### 1️⃣ **顶部统计卡片** - 一眼看懂业务状态

```
┌─────────────────────────────────────────────────────────┐
│  📊 Dashboard Overview                                  │
├─────────────────────────────────────────────────────────┤
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐│
│  │ 📩 新询价 │  │ 📧 已报价 │  │ ✅ 已成交 │  │ 📈 转化率 ││
│  │    12    │  │     8    │  │     3    │  │   25%   ││
│  │  +3 今天 │  │  待跟进  │  │  本月   │  │  本月   ││
│  └──────────┘  └──────────┘  └──────────┘  └──────────┘│
└─────────────────────────────────────────────────────────┘
```

**数据展示**：
- **新询价 (New)**: 总数 + 今天新增
- **已报价 (Quoted)**: 总数 + 待跟进提醒
- **已成交 (Closed)**: 本月成交数
- **转化率**: 成交数 / 总询价数

**交互**：
- 点击卡片 → 自动筛选对应状态的询价

---

### 2️⃣ **筛选和搜索栏** - 快速找到目标询价

```
┌─────────────────────────────────────────────────────────┐
│  🔍 [搜索邮箱/电话...]  📅 [日期范围]  🏷️ [状态筛选]   │
│                                                         │
│  窗户类型: [全部 ▼]  材料: [全部 ▼]  排序: [最新 ▼]    │

```

**筛选选项**：
1. **状态筛选**: 全部 / 新询价 / 已报价 / 已关闭
2. **日期范围**: 今天 / 本周 / 本月 / 自定义
3. **窗户类型**: Casement / Sliding / Double Hung / 等
4. **材料**: Vinyl / Aluminum / Wood
5. **排序**: 最新优先 / 最旧优先 / 金额高低

**搜索**：
- 支持搜索客户邮箱、电话号码


4. **视觉层次** - 重要信息突出显示

## 📊 数据库查询示例

### **获取统计数据**
```typescript
// 新询价数量
const { count: newCount } = await supabase
  .from('quotes')
  .select('*', { count: 'exact', head: true })
  .eq('status', 'new')

// 今天的新询价
const today = new Date().toISOString().split('T')[0]
const { count: todayCount } = await supabase
  .from('quotes')
  .select('*', { count: 'exact', head: true })
  .eq('status', 'new')
  .gte('created_at', today)

// 本月转化率
const thisMonth = new Date().toISOString().slice(0, 7)
const { data: monthlyQuotes } = await supabase
  .from('quotes')
  .select('status')
  .gte('created_at', thisMonth)

const total = monthlyQuotes.length
const closed = monthlyQuotes.filter(q => q.status === 'closed').length
const conversionRate = (closed / total * 100).toFixed(1)
```

### **获取询价列表（带筛选）**
```typescript
let query = supabase
  .from('quotes')
  .select('*')
  .order('created_at', { ascending: false })

// 状态筛选
if (statusFilter !== 'all') {
  query = query.eq('status', statusFilter)
}

// 日期范围
if (dateRange) {
  query = query.gte('created_at', dateRange.start)
                .lte('created_at', dateRange.end)
}

// 搜索
if (searchTerm) {
  query = query.or(`customer_email.ilike.%${searchTerm}%,customer_phone.ilike.%${searchTerm}%`)
}

// 分页
const { data, error } = await query
  .range(page * pageSize, (page + 1) * pageSize - 1)
```

## 🎯 用户体验流程

### **典型工作流程**

```
早上 9:00 - 查看新询价
├─ 打开 /dashboard
├─ 看到 "新询价: 3 个 (+3 今天)"
├─ 点击 "新询价" 卡片
└─ 自动筛选出 3 条新记录

处理询价
├─ 点击第一条记录查看详情
├─ 复制客户邮箱
├─ 在邮箱客户端回复报价
├─ 回到 Dashboard
├─ 点击 "⋮" → "更新状态" → "已报价"
└─ 添加备注："报价 $1500，等待客户回复"

下午 3:00 - 客户确认
├─ 客户电话确认接受报价
├─ 在 Dashboard 中找到该询价
├─ 更新状态为 "已关闭"
└─ 添加备注："客户接受，预计下周安装"

周五下午 - 查看周报
├─ 筛选日期范围：本周
├─ 查看统计卡片
└─ 本周：15 个询价，3 个成交，转化率 20%
```
